Bienvenido a la API de Estudiantes de CRUD Rápido

A continuación te explicamos cómo acceder a los endpoints:

1. Obtener todos los estudiantes (GET)

Ruta: /api/v1/estudiante/

Descripción: Este endpoint te devuelve la lista de todos los estudiantes registrados.

En Postman:

  • Método: GET
  • URL: /api/v1/estudiante/

2. Obtener un estudiante por ID (GET)

Ruta: /api/v1/estudiante/{id}

Descripción: Obtén los datos de un estudiante específico usando su ID.

En Postman:

  • Método: GET
  • URL: /api/v1/estudiante/{id}
  • Reemplaza {id} con el ID del estudiante que deseas consultar.

3. Guardar un nuevo estudiante (POST)

Ruta: /api/v1/estudiante/

Descripción: Usa este endpoint para guardar un nuevo estudiante.

En Postman:

  • Método: POST
  • URL: /api/v1/estudiante/
  • En el cuerpo (Body):
    • {
    "nombre": "Juan",
    "apellido": "Pérez",
    "edad": 20,
    "email": "juan.perez@mail.com"
}

4. Eliminar un estudiante por ID (DELETE)

Ruta: /api/v1/estudiante/{id}

Descripción: Elimina un estudiante mediante su ID.

En Postman:

  • Método: DELETE
  • URL: /api/v1/estudiante/{id}
  • Reemplaza {id} con el ID del estudiante que deseas eliminar.

Cada uno de estos endpoints requiere que tengas configurado correctamente tu entorno local, con el servidor ejecutándose en .
